# Platform-Engineering-for-Architects
Dieses Buch dient als zentrale Grundlage meiner Masterarbeit zu Platform Engineering in der Robotik. Es vermittelt praxisnahe Konzepte für den Aufbau skalierbarer, nutzerorientierter Plattformen, insbesondere durch Self-Service, Kubernetes und produktorientiertes Design, die sich auf robotische Systeme übertragen lassen.
